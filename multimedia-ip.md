# 🎤 多媒體IP・課程銷售簡報實戰（面談 close 簡報怎麼設計才會成交）

赫綵營銷部教材：賣「平面三階課程（平面基礎 → 平面進階 → 設計提案／IP 變現）」這條主線時，人坐下來後那套讓他「相信、想學、願意報整套」的簡報怎麼做。內容萃取自兩位選手真實競賽簡報＋李總等評審逐字回饋。

💡 一句話定位：簡報會不會成交，不在「漂不漂亮」，在「學生信不信他學完會變成那個樣子、願不願意買整套」。

---

## 🏆 評審回饋萃取的 10 大鐵則（核心）

1️⃣ **TA 要先設定，結尾要扣回來**：開場就講「我在對誰說話」（他在做什麼、卡在哪、想要什麼）→ 中段案例都對應這個 TA → 結尾扣回「這整套就是為你設計的」。念庭被點名唯一沒設 TA，案例帶得進、收不回。

2️⃣ **課程銜接邏輯（兩人共同最大弱點）**：每一階交接都要回答三問——①前一階給你什麼地基 ②這一階多解決什麼 ③兩階產出差在哪（最好並排比）。例：平面的 CIS 跟進階的 CIS 差在哪沒講，學生就會「只買第一階」。

3️⃣ **價值轉換要有目的性**：展示作品時每件補一句「為誰做／解決什麼／達成什麼目的」。把終點講成學生的未來（接案、就業、IP 變現、二代轉型），不是一堆好看的圖。

4️⃣ **用量化素材建立信任**：班級幾人滿班、產出幾件作品、師資具名（XX 是 UI 設計總監）、外部得獎、JOB 媒合 1111… 數字＋具名才有重量。

5️⃣ **講者台風＞簡報文字**：簡報是輔助，不是講稿。一頁一個重點，少字，靠口條補完。看到整段文案學生會自動略過。

6️⃣ **視覺用整張畫面、換不同學員作品**：學員作品／上課花絮滿版整圖、多位學員輪播，證明「不是只有一個成功案例」。

7️⃣ **成功故事要落地、可實現、感人**：講一個「和 TA 處境相似 → 學完 → 真的實現了什麼」的真人故事（餃子店二代做「攪拌」IP、二代接班創營收、日本人來台開店），比十張作品圖動人。

8️⃣ **金課獎包裝成「赫綵專屬舞台」**：重點不是 80 萬獎金，是「一個安全、專屬、能見證自己成長的舞台」，順勢邀約「今年一起投稿」。

9️⃣ **陪跑要講具體**：別只說「陪你成長」。量化成「我已幫你想好未來九個月／幾年怎麼走：打底 → 進階 → 提案 → 輔導接案／就業」，給有時間軸、有終點的路徑。

🔟 **選曲邏輯＋一場講透一門課**：三階跨度太大（平面→繪畫→IP）會繞、抓不到主軸。舉名 IP（Hello Kitty）要把它的週邊×課程內容結合著講，別只搬名牌反而襯托「追不上」。一場把一門課講透就好，可參考「米雅」流程整理動線。

## 🦴 理想銷售簡報骨架（7 段，每換一階做「銜接三問」）

1️⃣ 開場・立 TA：「你現在是 ___，卡在 ___，想要 ___」讓聽眾對號入座。
2️⃣ 三階整合路徑＋每階差異：平面基礎 → 平面進階 → 設計提案／IP 變現。
3️⃣ 學員作品＋上課花絮（整張畫面、多人輪播），每件補「為誰做、解決什麼」。
4️⃣ 落地成功故事（和 TA 呼應、有具體結果）。
5️⃣ 價值轉換＋量化信任＋陪跑（終點＝你的未來；帶數字具名；陪跑量化）。
6️⃣ 競賽舞台：金課獎＝專屬、為你設計的見證舞台 → 邀約投稿。
7️⃣ 收尾・把 TA 扣回：「所以這整套，就是為你設計的完整規劃」。

## 👥 兩位選手案例對比（學長姐示範）

🅰️ **奇奇《全能設計師養成計畫》**（平面＋進階＋提案）
被肯定：三階銜接相對完整、視覺清楚、有點出「把個人價值推銷出去」這個終點。
要改：口語有點繞；平面 CIS 與進階 CIS 差別沒講（學生覺得第二科可不上）；攝影／分組優勢沒在自己段落放大；陪跑、金課獎過場式不夠完整；畫面文字偏多。

🅱️ **念庭《從工具到變現：數位設計三階段》**（工具 → Procreate/IP → 變現）
被肯定：台風口語清晰、風格明確；IP／學生案例帶入聰明有畫面；「不會畫畫也能用 iPad」化解抗性處理得好。
要改：唯一沒設 TA（案例收不回，最被點名）；三階跨度太大主軸繞；舉名牌 IP 反襯「追不上」；簡報太多字、作品沒換多人。

🔴 **兩人共同通病**：①課程銜接／每階差異講不夠 ②金課獎／陪跑包裝不完整 ③簡報文字過多、作品沒用整張多人。

## ✅ 上場前自檢清單

• 開場 30 秒內讓聽眾對號入座（TA 具體背景）？
• 結尾有把 TA 扣回來？
• 每一階交接都回答了「地基／多解決什麼／差別」三問？
• 每件作品都補了「目的性／為誰做」？
• 有沒有至少一個「和 TA 相似、有具體結果」的落地故事？
• 班級／師資／得獎有沒有用數字＋具名？
• 陪跑有沒有講多久、到哪個終點？
• 金課獎有沒有包裝成「專屬、為你設計的舞台」＋邀約投稿？
• 作品／花絮是不是整張畫面、多位學員輪播？
• 頁面文字夠少、讓口條主導？整場主軸清楚（沒繞）？

---

## ⛔ 不該做的事

• 開場不設 TA → 後面案例再好都收不回。
• 一頁塞滿文案讓學生讀 → 他會略過，魅力是「看你說話」。
• 只放一位學員、只放漂亮圖不講目的 → 像「教你做漂亮的東西」而非解決問題。
• 階與階之間不講差別 → 學生會拆套、只買第一階。
• 舉名牌 IP 卻不接回課程 → 反襯「追不上」。
• 陪跑、金課獎含糊帶過 → 強賣點被講成過場話。
• 三階硬全塞、主軸繞 → 不如一場講透一門課。

## 一句話總結

> 公式：開場立 TA → 每階講清差別 → 作品講目的 → 落地故事 → 量化信任＋具體陪跑 → 專屬舞台 → 結尾扣回 TA。
> 兩大通病要避開：①銜接／每階差異講不清（學生拆套）②簡報太多字、作品沒用整張多人（沒說服力）。
