# 職涯出路 — 興趣也能成為工作

> 兩份簡報共同訴求：把攝影從興趣變成職涯新賽道。對應 104 上實際職缺類型。

## 104 職缺類型（學完可投遞方向）

簡報以 104 職缺牆呈現「攝影技能對應的真實工作機會」，涵蓋台北 / 台中 / 桃園等地，職缺類型包括：

- **人像攝影師**（工作室、婚紗、寫真）
- **平面 / 商業攝影師**（電商、商品、型錄）
- **影像後製 / 修圖師**（Lightroom、Photoshop）
- **社群小編 / 自媒體編輯**（內容產出、短影音）
- **行銷企劃**（結合視覺與社群思維）
- **攝影助理 / 棚務**

## 轉職與斜槓方向（CAREER GROWTH）

1. **職場新賽道** — 培養符合市場需求的行銷企劃、自媒體小編或專職攝影師能力，開啟職涯新可能。
2. **跨領域優勢** — 結合技術、美感與社群思維，在求職或面試時擁有獨一無二的視覺競爭力。
3. **學員實績** — 眾多學員成功將視覺技術應用於生活或轉職，實現從零到一的專業蛻變。

## 多元變現管道

- 接案（個人寫真、閨蜜寫真、商品、美食）
- 租棚接案 / 外包接案
- 圖庫上傳（被動式收入）
- 自媒體 IP（廣告、業配、課程、服務體系）
