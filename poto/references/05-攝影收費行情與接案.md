# 攝影收費行情與接案（結業變現）

> ⚠️ 以下是**學員結業後對外接案的市場行情**（以魚小魚攝影工作室為範例），用來證明「學費賺得回來」。
> **這不是上課學費**。學費請用 `hegrn-fee-table`（手機班 40,600 / 商業攝影班 49,250 / 自媒體班 32,400）。

## 寫真接案行情（範例：魚小魚攝影工作室）

| 方案 | 價格 | 內容 |
|---|---|---|
| 個人寫真 | **18,000** | 精修照片 ×15 張、轉風攝影棚、當日毛片全數提供、散髮造型 ×1、全程 2 小時 |
| 閨蜜寫真 | **24,000** | 精修照片 ×15 張、轉風攝影棚、當日毛片全數提供、散髮造型 ×2、全程 2 小時 |

### 加購行情

| 項目 | 價格 |
|---|---|
| 每加一小時 | 3,000 / 時 |
| 白禮服 / 晚禮服 | 8,500 / 件 |
| 精修照片 | 500 / 張 |
| 12 吋相本 | 3,500 / 本 |

> 拍攝地點：魚小魚攝影工作室；工作室保留活動之解釋、停止、終止等權益。

## 你也可以 —— 租借攝影棚接案

結業後可**租借專業攝影棚**自行承接案件，把上課學到的棚拍技術直接變現，不需自購昂貴設備與場地。

## 外包接案

- 透過外包平台、社群、工作室合作承接案件。
- 從圖庫被動收入、商品 / 美食拍攝、人像寫真到品牌型錄，逐步累積案源與口碑。

## 業務換算話術（範例邏輯）

> 「一場個人寫真接案行情 18,000，閨蜜寫真 24,000。換句話說，結業後接幾場案子，學費就回來了 —— 之後都是淨賺，而且作品集、攝影棚、師資人脈都還在。」
（實際換算請搭配當期學費與學員狀況，勿做收入保證。）
