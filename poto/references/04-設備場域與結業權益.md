# 頂級設備、專業場域與結業權益

> 兩份簡報共同的「硬體差異化」賣點：實體手把手 ＋ 業界級設備場域 ＋ 結業後可持續使用的資源。

## 頂級設備與場域

| 項目 | 說明 |
|---|---|
| 多樣式攝影棚 | 棚拍與實景佈光兼備，模擬真實商業拍攝環境 |
| 專業棚燈 | 商業人像用光、燈光配置實戰 |
| 專用化妝台 | 模特兒造型專用 |
| Apple 頂級工作站 | Apple 授權配置 iMac，後製體驗媲美業界標準 |
| 單眼租借 | 學員可租借單眼設備練習與接案 |
| 攝影道具 | 情境、商品、人像道具齊備 |
| 專業模特兒實拍 | 與真實模特兒合作演練，累積商業人像實戰經驗 |

> 設備區塊參考價值約 NT$17,850（Business Section）。

## 結業四大權益

1. **紀錄** — 完成獨一無二的個人作品集（一對一作品集指導）。
2. **競賽** — 參加國外或金赫獎比賽。
3. **參加** — 結業後參加 **6 個月外拍主題活動**。
4. **使用** — 租借專業攝影棚（可用於後續接案）。

## 賣點連結

- 「實體面對面、手把手教學」是相對線上免費影片的核心差異。
- 結業權益讓學習不止於下課：作品集 → 競賽得獎 → 外拍累積 → 租棚接案，形成完整變現閉環。
