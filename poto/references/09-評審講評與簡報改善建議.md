# 評審講評與簡報改善建議（教練用）

> 來源：兩段銷售簡報比賽的評審即時講評。整理成可直接拿來訓練、覆盤業務簡報的檢核清單。
> 用途：主管帶業務做簡報演練覆盤時，對照本檔逐項檢查。

---

## 一、盈偉版簡報 — 評審回饋

**做得好的地方**
- 科普式的鋪陳與結尾立意不錯，整體題材選得適合。
- 用「攝影比賽 / 國際賽事」拉高度，有記憶點。

**要改善的地方**
1. **重點收尾不完整**：解說從一段內容馬上接到下一段新內容，中間沒有停頓、沒把上一個重點收乾淨，聽眾感覺「重點還沒解決就跳走」。→ 每段講完要停一下、收束重點再往下。
2. **影片與口說搶注意力**：影片播放時台下需要做筆記、感受上「對接很遙遠」。若影片有聲音、你又要講話，台下不知道該聽你解釋還是看影片。
   - **技巧**：影片有聲音時，要講話就先把**音量轉小**（提示台下「請聽我說」），講完再轉回去，並說「老師示意大家繼續看」。用音量主導節奏。
3. **真人見證要有「歷程」**：講「某學生經營 IG 很成功」時，評審會想：他是誰？為什麼講他？是真實還是網路找的？
   - **銷售講故事公式**：什麼時候 → 發生什麼事 → 最後結果是什麼（從什麼變成什麼）。要在短時間內把「轉變歷程」講清楚，才有說服力。
4. **課程含金量沒講透**：「學生可以租棚、用學院設備環境賺錢」是很強的賣點，但講得不夠清晰。
5. **簡報頁面與口說不同步**：有幾頁「你已經講到了，但對應頁面還沒翻到」；成功案例 / 爆款頁只停 2 秒就跳過。→ 讓畫面先呈現、讓評審看完，再聽你解釋，節奏才對。
6. **現場燈光控制**：開場想關掉前面的東西卻找不到燈在哪。簡報前要先掌握場地燈光、投影、音響的控制位置。
7. **結尾的有獎徵答遊戲**：氛圍上能讓大家休息、感受比賽現場很好；但「群眾遊戲」風險是你無法控制台下，萬一觀眾亂給分會造成落差。要想清楚遊戲的用意（考試？氛圍？）。

## 二、韋頡版簡報 — 評審回饋

**做得好的地方**
- 三門課（手機 → 單眼 → 自媒體）的**包裝與銜接順暢**。
- 用影片帶出痛點、再破解痛點，是很棒的呈現方式。
- 介紹「誰來教（師資）」時點到位，講阿包、小魚老師的特色講得不錯。
- 反對問題環節用「工作 / 實務」來化解，方向對。

**要改善的地方**
1. **影片畫面焦點不清**：影片裡同時出現多個主體（例：一鍋裡兩隻互相打），台下不確定要看哪一個。畫面要聚焦單一主體。
2. **文字太多、缺對比範例**：影片 / 投影裡字可以**少一點**，多加**範例對比**（before/after）會更有說服力。
3. **念稿問題**：專業段落比較多是「念稿」，少了臨場表現。手機 / 單眼 / 自媒體三段**盡量用「表現（demo）」帶**，而不是一直念上面的字解釋差異。
4. **「作品集」沒講清楚**：一直說可以做作品集，但「作品集到底長什麼樣、為什麼要做、什麼才算好作品」沒講明白。要把作品的樣子具體呈現。
5. **口頭禪「可能」傷可信度**：整場一直「可能怎麼樣、可能有這個老師…」。銷售時「可能」這兩個字盡量不要出現，會顯得不確定、不專業，讓新客戶覺得「你好像沒那麼了解、像在唬我」。
6. **三門課的「選擇路徑」沒收束**：介紹完三門課後，學生最在意的是「我該從哪門開始？為什麼最後要選自媒體？」
   - 建議講清楚**循序漸進路徑**：手機入手 → 慢慢接觸商業（單眼）→ 自媒體上字幕 / 接案變現。讓人知道「我現在在哪個階段、該怎麼選」。
   - 把「靜態 → 動態」的**課程價值轉換模式**講清楚（因為內容多、講太快，這段被略過了）。
7. **反對問題「學不會怎麼辦」分數偏低**：前半段（會耐心教、可重修）沒問題，但缺少**後半段的保證機制**——
   - 客戶真正想聽的是：「**為什麼我來這裡一定學得會？**」
   - 要給出具體理由：課程規劃為什麼能讓你學會、復訓 / 換老師的機制、學會的比例 / 實績是多少。
   - 這個「保證學會的說法」，才是學生願不願意報名的關鍵。

## 三、通用簡報檢核清單（兩人共同問題彙整）

| 檢核點 | 標準 |
|---|---|
| 影片焦點 | 單一主體、不讓台下分心猜要看哪個 |
| 影片 vs 口說 | 不同時搶注意力；要講話先轉小音量 |
| 文字量 | 投影字少、改用範例 / before-after 對比 |
| 念稿 | 用表現 demo 取代念稿 |
| 重點收尾 | 每段講完停頓、收束再往下 |
| 真人見證 | 講清楚「歷程」：何時→發生什麼→變成什麼結果 |
| 口頭禪 | 戒掉「可能」「應該」等不確定字眼 |
| 課程路徑 | 收束成循序漸進路線，告訴客戶從哪開始、為何這樣選 |
| 反對問題 | 不只安撫，要給「保證學會」的具體機制與理由 |
| 簡報同步 | 口說與頁面同步，先讓畫面呈現再解釋 |
| 場控 | 事前掌握燈光 / 音響 / 投影控制位置 |
| 收尾 | 結尾的價值要講完整，不要戛然而止 |
