# 🎵 音樂課程全系列（音樂製作・詞曲創作・職業歌唱・全 AI 音樂）

赫綵設計學院的音樂產品線。整合陳怡雯（音樂本科）的「音樂製作 plus」銷售簡報，組成一條「從 0 到一首完整歌曲、再到舞台」的學習路徑。

💡 核心一句話：你聽一首歌很有感覺，是因為它打中你；那如果這首歌是你自己寫的故事呢？我們教你把「喜歡聽歌」變成「會做歌」。

## 🎯 四門課快速定位

🎹 **音樂製作人班（72hr）** — 用電腦做音樂（Logic Pro）、編曲、人聲錄製、完整製作流程。適合想自己從無到有做一首歌的人。

✍️ **詞曲創作人班** — 樂理（順行逆行、押韻、韻腳）、填詞、作曲邏輯。適合想寫歌寫詞、讓創作有依據的人。

🎤 **錄音室職業歌唱班（Live 星光歌唱培訓班・36hr）** — 發聲、咬字、換氣、情緒表達、錄音室實作與編修。零基礎愛唱歌即可，或修過音樂製作人課程者。

🤖 **全 AI 音樂創作班（42hr）** — 用 AI 工具加速音樂創作，系列進階。

🧩 三門核心課可串成一條路：寫詞曲 → 做編曲錄音 → 進錄音室唱，最後上舞台發表。

💰 學費請查費用表（音樂製作人約 42,300／Live 星光歌唱約 31,880／全 AI 音樂約 38,500，以現行合約為準）。

## ✨ 為什麼現在要學

🎚️ 一人分飾多角的時代：以前流行樂從作詞、作曲、錄音、混音、出版到行銷，每段都是不同專業；現在自媒體發達，一個人就能自己寫、自己錄、自己發到串流平台。

🤖 AI 與電腦降低門檻：用 Logic Pro 內建的大量音效與資料庫，老師會教你做出完整旋律，零基礎、不懂樂理也能入門。

🎓 實體手把手＋錄音室：在錄音室裡你才聽得清楚自己哪裡要調整，老師當場修正。

## 🎼 一首歌怎麼做（務必對接到課程）

➡️ 流程：作詞作曲（從 0 發想）→ 做 DEMO（Logic Pro）→ 編曲、伴奏（加鋼琴等配器）→ 人聲錄音 → 混音（讓作品變專業）→ 完整作品。

🔗 對接課程（業務一定要這樣講，學員才知道「跟著你，從這學到這，要花多少」）：

✍️ 作詞、作曲、樂理 → 詞曲創作人班
🎹 DEMO、編曲、伴奏、Logic Pro、人聲錄製、混音 → 音樂製作人班
🎤 演唱詮釋、發聲、咬字、換氣、情緒表達 → 錄音室職業歌唱班
🤖 AI 輔助創作貫穿全程 → 全 AI 音樂創作班

## 🎧 用「曲風」破冰（先問學員喜歡聽什麼）

❤️ 情歌：敘述感情、情緒多，最容易共鳴（聽到會想哭那種）。最大眾的入門切點。

🎼 Cover（翻唱／改編）：把喜歡歌手的歌改成自己的版本。可舉 AI 創作歌手「大頭針」。
⚠️ 講 Cover 要白話解釋，不然有人以為是「封面設計」。

🎛️ EDM／電音：想當 DJ、偏電音；會帶到旋律、loop、罐頭音效。

🎤 K-pop／嘻哈：像頑童 MJ116；要學 beat、節拍、節奏。

## 📚 各課程在教什麼

🎹 **音樂製作人班**：把生活靈感記下來 → 放進音樂軟體編曲 → 修一遍變雛形 → 加上你的創作變完整歌曲。帶你走完一首個人單曲，甚至從 EP 發起。

✍️ **詞曲創作人班**：填詞（主歌、副歌、橋段）、詞曲曲賦，讓作曲有邏輯；樂理基礎讓創作有依據；再發展到編曲與錄音操作。

🎤 **錄音室職業歌唱班**：很多人想唱好卻不知怎麼詮釋。老師教咬字、換氣（什麼時候換氣、怎麼表達情緒），從發聲訓練到錄音室實際錄音、編修回看，最後有錄音室作品可發表。

## 🙋 常見反對問題（QA）

❓ 不懂樂理、不會看譜，只是喜歡唱歌，適合嗎？
✅ 適合。現在不像以前一定要會作曲；用 Logic Pro 的內建資料庫，老師帶你做出完整旋律，搭配有故事性的詞曲，一首歌不難做出來。

❓ 為什麼要來你這上課？你能教我什麼？
✅ 講明確終點＋含金量：你會從 0 學會作詞作曲、用電腦／AI 編曲、進錄音室錄製混音，產出一首可發行、可上舞台的個人作品；再用師資年資、學生成果、設備場域把含金量講具體。

❓ 網路免費影片就能學，為什麼付費？
✅ 系統化從 0 到成品的路徑與陪跑、錄音室與設備場域、師資業界資歷、舞台與發行機會（學院資源，個人拿不到）。佐證要用學院實證，不要用網路素材。

❓ 我不擅長唱歌，舞台／發片跟我無關吧？
✅ 不是。舞台是合作型機會，作詞作曲＋演唱都算；團隊分工有人 Rap、有人唱、有人當 Leader，完成課程就能參與。
